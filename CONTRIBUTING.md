# Contributing

Thank you for helping improve the Blueprint Functional Health Wiki.

## How to contribute

1. Fork the repository.
2. Create a branch: `git checkout -b my-improvement`
3. Add or improve Markdown files inside `docs/`.
4. Cite reputable scientific sources when possible.
5. Submit a pull request.

## Contribution standards

- Keep content educational, not prescriptive medical advice.
- Avoid diagnosis or treatment claims.
- Cite peer-reviewed or reputable sources when possible.
- Clearly label uncertainty or emerging evidence.
- Use plain language when possible.
